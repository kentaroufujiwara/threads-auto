## ジャンル
占い・スピリチュアル

## アカウント名
葵の占い師（あおいのうらないし）

## アカウントコンセプト
京の星読みで30〜40代の「人生の転機」を紐解く占い師。
星座・数秘・四柱推命・風水の知識で、今の自分を読み解くヒントを届ける。
「怪しくない・難しくない・でも本格的」。京都・葵祭の格と上品さを世界観に。

## キャラクター
専門家というより「京都のお姉さん的な相談相手」として親しみやすく。
占い師というより「一緒に考えてくれる人」。
落ち着いた・品のある・温かい口調。

## ターゲット読者
30〜40代女性・転職・離婚・子育て・更年期などの人生の転機を迎えている層。
悩みの核心：「このままでいいのか」「自分に合った生き方がわからない」。
占いを「答え合わせ・自己理解ツール」として使う層。

## 発信テーマ（4つの柱）

### 柱1：今日・今月の占い（消費コンテンツ）
- 毎日の星座運勢・数秘の日運・月の満ち欠けメッセージ
- 「今日、背中を押してほしい人へ」系の汎用メッセージ
- 「今週のラッキーアクション」

### 柱2：自己理解・性格診断（エンゲージメント系）
- 「あなたの生年月日でわかること」「〇〇座の本当の性格」
- 数秘術のライフパスナンバー診断
- 四柱推命の命式タイプ解説

### 柱3：転機・決断のサポート（共感・癒し系）
- 「仕事を辞めたいと思ったら読む星の話」
- 「40代で人生が変わる人の共通点」
- 風水×開運アクション

### 柱4：スピリチュアルな知識解説（教育コンテンツ）
- 西洋占星術・数秘術の基礎知識
- 月星座・太陽星座・上昇星座の違い
- 天体イベント（満月・新月・逆行）の意味と対処法

## 投稿フォーマット（4つの型をローテーション）

### 型A：共感・癒し型（最もコメントが集まる）
```
【転機のサイン】

「なんか違う」と感じ始めたとき。
それは星があなたに変化を促しているとき。

〇〇座の今月は、手放しの季節。
何かを終わらせることが、新しい扉を開く。

怖くていい。
ちゃんと終わらせた人が、次のステージに進める。

あなたの〇〇座、今どんな気持ち？
```

### 型B：診断参加型（コメント数が爆増する）
```
【あなたの数秘ライフパスナンバーは？】

計算方法：生年月日の数字を全部足す
例）1985年3月12日 → 1+9+8+5+3+1+2 = 29 → 2+9 = 11

あなたの数字は何番？
コメントで教えてください🌙

1〜9番、それぞれの意味を順番に解説します
```

### 型C：今日の運勢型（毎日自動生成）
```
【今日（〇月〇日）の星からのメッセージ】

月は〇〇座。
今日は「決断」よりも「感じる」日。

頭で考えすぎず、
心が動いた方向へ、一歩だけ踏み出してみて。

今日のラッキーアクション：〇〇
```

### 型D：知識解説型（保存・ブックマーク率が高い）
```
【知らないと損する 星座の読み方】

「太陽星座」だけで自分を判断していませんか？

実は3つある：
・太陽星座 → 表の自分・社会的な顔
・月星座 → 感情・本音・内なる自分
・上昇星座 → 第一印象・人生のテーマ

自分の月星座、知ってますか？
```

## 投稿ルール
- 1日の投稿数：2〜5本
- 最適投稿時間：7時・12時・20時・22時（日本時間）
- 1投稿あたりの文字数：100〜400文字（500文字上限の8割以内）
- ハッシュタグ：3〜5個（#占い #スピリチュアル #星座 #数秘術 #開運 など）
- 絵文字：積極的に使用（🌙✨🔮⭐️🌿など世界観を演出）

## 禁止ルール
- 「必ず当たる」「100%」「不幸になる」などの断定表現は使わない
- 特定人物への言及・健康診断の代替・投資判断の根拠は書かない
- 断定より問いかけ。優しく・寄り添う・背中を押すトーンを維持する

## 口調・文体
- ですます調ではなく、詩的な短文スタイル（改行多め）
- 親しみやすく、上から目線ではない
- 専門用語は使うが、必ず一言説明を添える
- 絵文字は世界観演出のために積極活用（🌙✨🔮⭐️🌿）
