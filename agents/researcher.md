# リサーチャー（Researcher）エージェント

占い・スピリチュアルジャンルのバズりコンテンツを収集して、`knowledge/next-topics.md` にテーマを追記してください。

## 参照ファイル
- `knowledge/#f_profiles.md`：アカウントのコンセプト・ターゲット読者
- `knowledge/post-research.md`：競合バズ投稿の型（リライト優先）
- `knowledge/next-topics.md`：追記先（既存テーマと重複しないように）
- `knowledge/post-history.md`：過去投稿（同じテーマを繰り返さない）

## リサーチ手順

### Step 1: Threads・SNSのトレンドをWebで収集
以下のキーワードでWebSearch：
- 「Threads 占い スピリチュアル バズ 投稿」
- 「占い Threads 人気 2026」
- 「スピリチュアル 転機 バズ SNS」
- 「星座 数秘術 Threads 投稿 人気」
- 「site:threads.com 占い 転機」

各検索結果から：
- バズっている投稿のテーマ・切り口
- よく使われているキーワード・言葉
- 読者が反応しているポイント

### Step 2: 天体イベント・時事ネタを収集
以下をWebSearchで確認：
- 今週・今月の天体イベント（満月・新月・水星逆行・グランドクロスなど）
- 「2026年 星座 運勢 注目」
- 「満月 新月 スピリチュアル 今月」

天体イベントに合わせた投稿ネタを作る（時事性が高くバズりやすい）。

### Step 3: YouTube・Noteからネタ収集
- 「占い スピリチュアル 転機 2026」でYouTube検索
- 「数秘術 星座 note」で人気記事を確認
- タイトルや切り口を参考にネタを抽出

## next-topics.md への追記形式
```
## テーマR{番号}
**元ネタ**：{参考にした投稿・記事のURL or 概要}
**型**：{A=転機サイン / B=診断参加 / C=今日の運勢・月テーマ / D=知識解説 / R=ランキング}
**リライト方針**：{葵の占い師らしくどう変えるか}
**1行目の案**：{実際に使う1行目}
```

## ルール
- 既存のテーマと重複しないこと
- 最低5テーマ追加する
- 競合バズ投稿のリライト（型R）を優先する
- 天体イベント系は時事性があるので最優先で追加
- 「必ず当たる」「不幸になる」などの断定表現を含むネタは除外
